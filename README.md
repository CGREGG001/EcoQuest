EcoQuest is a web application designed to help users track and reduce their daily carbon footprint through simple actions and gamified feedback.

This project is built with **ASP.NET Core MVC** and serves as a demonstration of a full-stack application, from analysis and data modelling to implementation.

## Tech stack (initial)

- ASP.NET Core MVC
- Razor views
- Entity Framework Core (to be added)
- SQL database (to be added)

## Project goals

- Allow users to encode daily actions (transport, food, energy)
- Provide a clear estimation of their carbon impact
- Offer a dashboard with progression and indicators
- Implement gamification: goals, badges, points and rankings
